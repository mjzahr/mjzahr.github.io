Computational Mechanics and Optimization Laboratory

autogen

A collection of Python utilities that provides a convenient interface to MAPLE, mainly for the purpose of code generation. Strings containing MAPLE expressions are built-up using Python strings and the utilities provided by autogen, which are passed to MAPLE for analysis and code generation. The library provides utilities for linear algebra, standard functions (Gaussian, rigid body motion, radial basis functions, Fourier series, blending maps), transformation of integrals/PDEs between domains, Arbitrary Lagrangian-Eulerian formulation of PDEs, conservation laws (fluxes, state variables, Riemann solvers, numerical fluxes, etc for Burgers' equation, the incompressible and compressible Navier-Stokes equations, among others). The main use of the library is to generate code for the derivatives of complex expressions needed by computational mechanics software, e.g., Jacobian matrices for nonlinear solvers or derivatives needed for PDE-constrained optimization.

Source code available on GitHub.

cclib

A collection of C/C++ utilities used by other codes in the Computational Mechanics and Optimization Lab (anlytc, cplib, femlib, riemlib, spgrid). Functionality includes flexible multidimensional array classes with binary I/O capabilities that use same binary format as NUMPY for seamless interface with Python, template classes to combine arbitrary array classes (with required API) into a set of array or concatenated array, sparse and block sparse matrix classes, template classes to combine arbitrary matrix classes into block matrices, timing utilities, interfaces into dense and sparse linear algebra software (BLAS, LAPACK, SuiteSparse), finite difference utilities to conveniently check derivative implementation, a generic optimization class with interfaces to commonly used optimization software (SNOPT, IPOPT), a collection of nonlinear solvers for systems and least squares problems, and a collection of tensor and simplex utilities useful in defining polynomial basis functions and quadrature rules over such elements.

Source code available on GitHub.

pylib

A collection of Python utilities used by other codes in the Computational Mechanics and Optimization Lab (autogen, pymortestbed, pytikz). Functionality includes flexible type checking, bookkeeping/database utilities, an extended string class that redefines basic operations +, -, *, /, etc to perform the appropriate arithmetic operation, e.g., 'a'+'b'='a+b', finite difference utilities to conveniently check derivative implementation, interfaces into ImageMagick and FFMPEG, image generation and postprocessing using MATPLOTLIB and Scipy, and a collection of tensor utilities.

Source code available on GitHub.

pytikz

A collection of Python classes and utilities to conveniently create TikZ and PGFPlots files for incorporation in LaTeX documents. This package allows the user to harness the power of Python to create sophisticated TikZ/PGFPlots images without needed to know the fine details of programatic TikZ/PGFPlots.

Source code available on GitHub.
